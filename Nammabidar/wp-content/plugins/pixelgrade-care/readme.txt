== Description ==

This plugin offers fully compatibility with Pixelgrade themes features like automatic updates, import demo data and more others features.

== Installation ==

1. Search and install Pixelgrade Care plugin through the 'Plugins' menu in WordPress
2. Install a Pixelgrade theme and create a Pixelgrade account, voila!
